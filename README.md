# Netflix Data Analysis Dashboard (Tableau Project)

##  Project Overview

This project is an interactive Netflix Dashboard created using Tableau Public.
The dashboard analyzes Movies and TV Shows available on Netflix based on Year, Genre, Country, Rating, and Type.

The goal of this project is to demonstrate Data Analysis, Data Visualization, and Dashboard Design skills required for a Data Analyst role.

---

##  Dashboard Features

* Total Movies & TV Shows by Year (Trend Analysis)
* Movies vs TV Shows Distribution
* Top 15 Genres
* Content by Country (Map Visualization)
* Rating Distribution
* Interactive Filters (Type, Rating)

---

##  Tools Used

* Tableau Public
* Excel / CSV Dataset
* Data Cleaning
* Data Visualization
* Dashboard Design
